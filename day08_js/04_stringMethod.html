가장 많이 다루는 데이터 타입 : string

  • 문자열 데이터는 index(0부터 시작)를 가지고 있음
  • 문자열 데이터 값 내에서는 개행(br)하는 것을 추천하지 않는다
  • new String(); => 객체 타입으로 선언은 가능하지만 추천하지 않는다
  • length : 문자열의 총 길이 => index로 변환할 때는 length-1
  • escape문자 문자 사이에 예약어를 삽입하려면 \

문자 관련 메서드 : java에서 사용하는 String 메서드와 비슷 (다른 프로그래밍 언어도 비슷)

  • 문자 추출 관련 메서드 : substring(), substr(), slice()
  • 문자 대체 메서드 : replace(oldStr, newStr) => 첫번째 매칭되는 문자만 바꿔줌, 대소문자 구분
  • trim() : 문자열 양측 공백 제거
  • padStart(목표문자길이, 채울문자) / padEnd(목표문자길이, 채울문자) : 앞쪽에서 / 뒷쪽에서 채우기
    문자값의 힌트 => 010101 - ******* , 서울시 ****
  • charAt(index) : 해당 index 문자 추출
  • indexOf("str") : 문자열의 위치를 반환
  • startWith("str") / endWith("str") : 지정 문자열이 탐색대상 문자열에서 시작/끝에 존재하는지 확인하는 메서드
    return => true / false 반환 ex) 확장자 확인
