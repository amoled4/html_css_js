CSS 적용은 element.style.property 형식으로 접근 및 변경이 가능

  • property는 자바스크립트의 코드 : -(하이픈) 적용 안 됨, 띄어쓰기 적용 안 됨
  • css의 속성의 -(하이픈)을 제거하고, 카멜표기법으로 변경
  • Element.setAttribute(name, value) : Element에 속성 부여 이미 있는 속성이면 업데이트, 없는 속성이면 새 속성으로 추가
  • CSSStyleDeclaration.setProperty(propertyName, value) : css 스타일을 부여할 대상에게 css 프로퍼티 부여함 propertyName : css 속성 이름, value : 속성 값

색상명을 입력하고, 색상 변경 버튼을 클릭하면 현재 h1 태그의 배경색이 입력한 색상이 되게끔 변경 style.backgroungColor X
